## Wordを初期設定に戻す(恒常的に)
1. Wordを終了する。
2. `C:\User\(ユーザー名)\AppData\Roaming\Microsoft\Templates`に移動する
3. 2で移動したディレクトリ内にある、`Normal.dotm`を`Normal_old.dotm`などとリネームする。
4. Wordを再起動する。

<a id="2"></a>
## PowerPointのデザインの提案をオフにする
1. ファイルタブに移動する。
2. オプションをクリックする。
3. 全般に進み、PowerPoint デザイナーの「デザインのアイデアを自動的に表示する」のチェックを外してOKをクリックする。

<a id="3"></a>
## PowerPointのプレースホルダー自動調整を無効化する
1. ファイルタブに移動する。
2. オプションをクリックする。
3. オートコレクトのオプションに進む。
4. 入力オートフォーマットタブから、「タイトル/本文のプレースホルダーに自動的に収める」のチェックを外す。

<a id="4"></a>
## Excelファイルにパスワードをかける
1. パスワードをかけたいファイルを開く。
2. ファイルタブの情報に移動する。
3. 一番上にある、ブックの保護を選択し、「パスワードを使用して暗号化」を選ぶ。
4. 自分で決めたパスワードを2回打つ。

<a id="5"></a>
## Wordなどで背景透過の画像を作成する
### 画像の背景が一色の場合
1. 編集したい画像をクリックして、図の形式タブにある色をクリックする。
2. メニュー内の透明色の指定をクリックする。
3. マウスポインタがスポイトになるので、透明にしたい部分をクリックする。

※ 画像内に背景と同じ色が使われている場合、透明になってしまうので、注意。

### 写真の背景の場合
1. 編集したい画像をクリックして、図の形式タブにある背景の削除をクリックする。
2. 背景が自動判別され、削除対象の部分が紫になる。
3. 残したい部分については、保持する領域としてマークをクリックして、クリックまたはドラッグ操作で選択し直す。
4. 削除したい部分については、削除する領域としてマークをクリックして、クリックまたはドラッグ操作で選択し直す。
5. 変更を保持をクリックする。
