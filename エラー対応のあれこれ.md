# 仮想環境上でpythonのライブラリをインストールすると、Conda SSLエラーが出る問題

## 問題

具体的には、`pip install`や`conda install`を特定の仮想環境上で行うと、下記のエラーが出る。
```
Conda SSL Error: OpenSSL appears to be unavailable on this machine. OpenSSL is required to download and install packages.
```

## 対処法
anaconda3>library>binにある以下のファイルを、anaconda3>DLLsにコピーして、再起動すると解決。
- libcrypto-1_1-x64.dll
- libssl-1_1-x64.dll
